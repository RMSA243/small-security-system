How to Run the Code:
........................................................................
Backend Setup:

	Install Python and Dependencies:

		Ensure Python 3.9 or later is installed.

		Install required packages:

			pip install flask flask-cors cryptography

	Start the Flask Backend:

		Navigate to the backend directory.

		Run the Flask application:

			backings\Scripts\activate :: TO ACTIVATE THE virtual ENV

			python main.py :: to run backend server

		The backend will start on http://127.0.0.1:8080.
.........................................................................

Frontend Setup:

	Install Node.js and Dependencies:

		Ensure Node.js 16 or later is installed.

		Navigate to the frontend directory and install dependencies:

			npm install


	Start the React Frontend:

		Start the development server:

			npm run dev

		Access the frontend at http://127.0.0.1:5173.


	Build for Production: (optional not ready yet)

		Build the React application:

			npm run build

		Serve the build files using a static server like Nginx.
